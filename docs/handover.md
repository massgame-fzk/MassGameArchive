# 引き継ぎメモ

このプロジェクトは、広大附属マスゲームの曲とYouTube動画を整理する非公式アーカイブです。

## 運用方針

- 公開対象は年度、部門、場面、曲名、アーティスト、YouTubeです。
- 個人名、写真、本文レポート、掲示板、メールアドレスは公開しません。
- YouTube以外の外部リンクは公開しません。
- 出典や確認経緯は管理用データやPull Request本文に残します。

## 役割

- 年度担当者: 担当年度の曲情報とYouTube情報を集め、Pull Requestを作成する。
- レビュー担当者: 内容、根拠、公開方針違反がないか確認する。
- 管理者: Pull Requestをマージし、デプロイ結果を確認する。

## よく触る場所

- `src/data/years.json`: 年度ページの説明
- `src/data/music.json`: 曲データ
- `src/data/media.json`: YouTubeデータ
- `docs/github-operation.md`: 操作手順
- `design.md`: デザイン方針

## 更新時の注意

- 迷った情報は公開データに入れず、IssueやPull Requestで相談します。
- YouTubeリンクはプレイリストまたは動画URLを使います。
- 新しい年度を追加したら、年度ページ、曲一覧、動画ページの3箇所を確認します。
- JSONのカンマ抜けや引用符ミスがあるとビルドに失敗します。必ず `npm run build` を実行します。
